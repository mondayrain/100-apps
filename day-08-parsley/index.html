HTML5

HTML5 comes with built-in form input types that are semantic.

Many browsers provide built-in validation for some input types like email and url.

Alternatively, HTML5 now allows "constraint validation", which lets browsers run attributes like "min", "max", "pattern", and "required" to figure out if the form input is valid _without_ JavaScript.

Non-HTML5 / Using JavaScript and Parsley.js

Parsley uses jQuery, and does validation through JS.

You can add validation by either 1) defining `data-parsley-validate` on your form or running `$('#parsley-form').parsley()` in a script, then 2) defining `data-parsley-#{some-attribute}=value` on your inputs. This should auto-magically validate input!
